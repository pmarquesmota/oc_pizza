Cas n°1

Nom : Suivre l'état des stocks (package « Gestion gestion administrative »)
Acteur(s) : administrateur
Description : L'administrateur doit pouvoir suivre l'état des stocks
Auteur : Paul Marques Mota
Date(s) : 14/10/19 (première rédaction)

Pré-conditions : authentification
Démarrage : L'administrateur a cliqué sur la page "suivre l'état des stocks"

DESCRIPTION

Le scénario nominal :

1. Le pizzaiolo clique sur "afficher l'état des stocks"
2. Le système cherche l'état des stocks
3. Le système affiche l'état des stocks

Les scénarios alternatifs

1.a Le pizzaiolo décide de quitter l'affichage

Fin : Scénario nominal : à l'étape 1 ou 3 sur décision de l’administrateur

Post-conditions : Aucun

COMPLEMENTS

Aucun
